# Project Context

**UniDeploy** is a security and deployment readiness scanner for vibe-coded applications.

## Current State
- Monorepo with Next.js frontend and FastAPI backend.
- Simple Authentication and DODO Payments integration completed.
- Custom A2A (Agent-to-Agent) memory bus and Google ADK engine currently in place (slated for removal).

## Target State
- Scalable multi-agent architecture using LangGraph.
- Seamless CLI-to-Dashboard WebSocket communication.
- Containerized for cheap/free cloud deployment.
