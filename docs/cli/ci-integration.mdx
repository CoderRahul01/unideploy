title: "CI/CD Integration"
description: "Gate pull requests on security findings using unideploy init --ci."

## Overview

UniDeploy's `--ci` flag exits with code `1` if any CRITICAL findings are present, blocking the merge. The scan runs fully offline — no session, no dashboard connection.

## GitHub Actions

```yaml
# .github/workflows/unideploy.yml
name: UniDeploy security scan

on:
  push:
    branches: [main]
  pull_request:

jobs:
  security:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Install UniDeploy CLI
        run: npm install -g unideploy

      - name: Run scan
        id: scan
        run: |
          unideploy init --ci --json > report.json || EXIT=$?
          cat report.json | jq '{grade: .summary.grade, total: .summary.total, critical: .summary.critical, risk_score: .summary.risk_score}' >> $GITHUB_STEP_SUMMARY
          exit ${EXIT:-0}

      - name: Upload report
        if: always()
        uses: actions/upload-artifact@v4
        with:
          name: security-report
          path: report.json

      - name: Comment on PR
        if: github.event_name == 'pull_request' && always()
        uses: actions/github-script@v7
        with:
          script: |
            const fs = require('fs');
            const report = JSON.parse(fs.readFileSync('report.json', 'utf8'));
            const s = report.summary;
            const body = `## UniDeploy Security Scan\n\nGrade: **${s.grade}** | Risk Score: **${s.risk_score}/100**\nFindings: ${s.total} (${s.critical} critical, ${s.high} high)\n\nView the full report in the uploaded artifact.`;
            github.rest.issues.createComment({ issue_number: context.issue.number, owner: context.repo.owner, repo: context.repo.repo, body });
```

## GitLab CI

```yaml
security-scan:
  image: node:20
  script:
    - npm install -g unideploy
    - unideploy init --ci --json > report.json
  artifacts:
    paths:
      - report.json
    when: always
```

## Bitbucket Pipelines

```yaml
pipelines:
  default:
    - step:
        name: Security scan
        image: node:20
        script:
          - npm install -g unideploy
          - unideploy init --ci
```

## Exit codes

| Code | Meaning |
|------|---------|
| `0` | Scan completed, no CRITICAL findings. |
| `1` | CRITICAL findings present — pipeline should fail. |
