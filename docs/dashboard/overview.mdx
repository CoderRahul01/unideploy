The UniDeploy dashboard at [unideploy.vercel.app](https://unideploy.vercel.app) is where you view scan results, track findings over time, and trigger auto-fixes.

## Two entry points

**GitHub URL scan** — paste a repository URL into the input on the dashboard home. The scan runs in the background and results appear within 30–60 seconds for most projects.

**CLI session** — run `unideploy init` in your terminal, enter the 6-digit code at [unideploy.vercel.app/connect](https://unideploy.vercel.app/connect), and findings stream to the dashboard in real time as the scan progresses.

## What the dashboard shows

- Security grade (A–F) with colour coding
- Findings table grouped by severity
- Per-finding remediation plans (paid plans)
- Auto-fix status and PR link (paid plans)
- Scan history per project
